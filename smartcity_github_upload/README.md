# 스마트시티 비교분석 시스템 (판교테크노밸리 vs. 인천 청라국제업무지구)

이 프로젝트는 수도권 최고의 성공 사례인 **판교테크노밸리**와 개발이 저조/지연되고 있는 **인천 청라국제업무지구**의 토지이용, 교통망(지하철 네트워크), 인구사회 지표를 정량적으로 비교 분석하여 업무지구의 성공 요인을 진단하는 웹 기반 시각화 시스템입니다.

---

## 1. 프로젝트 구조

```
smartcity_analysis_project/
├── data/                    # 전처리된 JSON/GeoJSON 데이터 파일

├── data.js                  # 인라인 데이터 패키지

```

## 3. 실행 및 재현 방법

### 데이터 전처리 실행 (Python)

1. 필요한 패키지 설치:
   ```powershell
   pip install geopandas pandas numpy scipy openpyxl
   ```
2. `smartcity_analysis_project/src/preprocess.py` 실행:
   ```powershell
   python src/preprocess.py
   ```
   이 스크립트는 원본 Shapefile들을 Leaflet 뷰어에 최적화된 EPSG:4326(WGS84) 좌표계로 자동 변환하고, 용도지역 비율, 토지이용 혼합도(LUM), 지하철 노선망 다익스트라(Dijkstra) 경로 최단 소요시간, 누적 도달가능 인구 등을 계산하여 `data/` 폴더 내에 정적 JSON 파일로 저장합니다.

### 로컬 웹 서버 실행 (옵션)

별도 서버/DB 구축 없이 정적 HTML 파일로 구동됩니다.

1. Python을 이용한 간이 웹 서버 구동 (프로젝트 루트 폴더에서 실행):
   ```powershell
   python -m http.server 8000
   ```
2. 웹 브라우저에서 아래 주소로 접속:
   ```
   http://localhost:8000/
   ```

## 4. 핵심 분석 지표 요약

| 분석 지표 | 판교테크노밸리 | 인천 청라국제업무지구 | 스마트시티 진단 및 해석 |
|---|---|---|---|
| **총 건축 연면적** | **4,753,473 ㎡** | **4,932,277 ㎡** | 청라는 판교와 부지 면적이 비슷함에도 실질적인 업무/상업 기능보다는 단순 주거단지가 대부분을 차지함 |
| **토지이용혼합도 (LUM)** | **0.845** | **0.690** | 판교는 다각화된 용도 혼합(업무+주거+상업)으로 주야간 활성화를 유도하나, 청라는 단일 용도 편중이 심함 |
| **업무용 연면적 비율** | **43.9 %** | **14.7 %** | 청라는 자족형 업무지구로서의 본래 기능보다 주거용(59.9%) 비중이 높은 "베드타운"화 현상 노출 |
| **30분 도달 지하철역 수** | **63 개역** | **39 개역** | 판교역(신분당선/경강선)의 광역적 허브 역할로 강남 및 성남 전역으로의 대중교통 접근성이 우수함 |
| **60분 도달 지하철역 수** | **436 개역** | **237 개역** | 서울·경기 남부 거대한 광역교통망 확보 측면에서 판교가 압도적 우위 |
| **60분 도달 광역 인구수** | **약 810만 명** | **약 313만 명** | 판교는 60분 내에 서울·경기 인구 810만 명에 접근할 수 있어 노동시장(Labor Pool) 확보에 절대적 유리함 |
| **60분 도달 광역 종사자수** | **약 557만 명** | **약 224만 명** | 기업 입주 수요와 직결되는 배후 종사자수에서 판교가 2.5배 이상 압도적인 차이를 보임 |
